# 💻 Tugas Study Group Chevalier Lab

Selamat datang di repository **TugasCheva**! Repository ini berisi kumpulan tugas Frontend Web Development yang dikerjakan sebagai bagian dari kegiatan pembelajaran di **Study Group Chevalier Lab**. Setiap tugas (Tugas 1 hingga Tugas 8) merupakan tahapan progresif yang dirancang untuk mempraktikkan dan meningkatkan keterampilan dalam pengembangan web.

## 📁 Struktur Repository

Repository ini dibagi menjadi beberapa direktori tugas (Tugas 1 - Tugas 8). Berikut adalah gambaran singkat mengenai materi dari setiap penugasan:

- **[Tugas 1](./Tugas1)**  
  Fokus pada dasar elemen **HTML** dan stlyling dasar menggunakan **CSS**.
- **[Tugas 2](./Tugas2)**  
  Penerapan HTML, CSS, dengan pengenalan **JavaScript (Vanilla JS)** untuk memberikan interaktivitas dasar pada antarmuka, khususnya terkait manipulasi DOM.
- **[Tugas 3](./Tugas3)**  
  Pengembangan dan styling form khusus (seperti halaman *Login/Register*) menggunakan file HTML dan CSS secara eksternal.
- **[Tugas 4](./Tugas4)**  
  Eksplorasi antarmuka responsif dan penerapan *Vanilla HTML/CSS/JS* yang lebih rapi dan mendalam.
- **[Tugas 5](./Tugas5)**  
  Melatih logika JavaScript tambahan dan pendemonstrasian komponen dinamis pada web page statis.
- **[Tugas 6](./Tugas6)**  
  Review tugas atau eksplorasi komponen pada file tunggal (HTML only / HTML with embedded CSS).
- **[Tugas 7](./Tugas7)**  
  Lompatan dari Vanilla Javascript ke library modern **React JS** yang dibuild menggunakan **Vite**. Praktik dasar mengenai pembuatan komponen (components).
- **[Tugas 8](./Tugas8/data-fetching)**  
  Puncak dari tugas-tugas sebelumnya, membuat aplikasi React berupa *katalog film* lengkap dengan manajemen routing aplikasi (`React Router`) dan *Data Fetching* asinkron untuk mengambil data antarmuka.

## 🚀 Cara Menjalankan Project

### Web Klasik (Tugas 1 - Tugas 6)
Untuk menjalankan tugas berbasis web klasik (HTML/CSS/JS):
1. Masuk ke folder dari tugas yang ingin Anda periksa.
2. Buka (`open with`) file `index.html` menggunakan *web browser* bebas.
3. Sebagai opsi lain, Anda dapat membukanya menggunakan ekstensi **Live Server* pada editor kode Visual Studio Code.

### SPA Modern (Tugas 7 & Tugas 8)
Untuk project web Single Page Application yang menggunakan **React** dan **Vite**:
1. Buka Command Line / Terminal di mana folder Node.js project berada (Misal: `cd Tugas8/data-fetching`).
2. Instal semua *dependency* yang diperlukan dengan menjalankan perintah:
   ```bash
   npm install
   ```
3. Mulai server pengembangan secara statis dengan menjalankan:
   ```bash
   npm run dev
   ```
4. Buka URL `http://localhost:5173/` (atau port sesusai terminal Vite) di browser untuk melihat aplikasi berjalan.
